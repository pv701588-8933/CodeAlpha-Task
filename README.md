1. Web Scraper Project

A Python-based web scraping tool that extracts book data from
https://books.toscrape.com
 and allows:
-Fetching book details
-Filtering by price
-Filtering by rating
-Searching by title
-Viewing description & UPC
-Saving scraped data to MongoDB using a Node.js backend

This scraper reads only publicly available data meant for educational practice.

Tech Stack
Backend (Database Layer)
-Node.js
-Mongoose
-MongoDB Atlas
-dotenv

Scraper
-Python
-Requests
-BeautifulSoup (bs4)
