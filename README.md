<h1>PROJECT 2. EDA [EXPLORATORY DATA ANALYSIS</h1>

<h1>Netflix Movies Dataset – Exploratory Data Analysis (EDA)</h1>
<h2>Project Overview</h2>

This project performs an in-depth Exploratory Data Analysis (EDA) on a Netflix Movies dataset.
The goal is to uncover trends related to genres, popularity, release years, voting patterns, and overall movie performance.

<h3>The analysis includes:</h3>

Data cleaning
Handling missing values
Feature understanding
Visual exploration
Insights and business-ready conclusions

<h2>Dataset Description</h2>
<h3>The dataset contains various movie attributes such as:</h3>

Title
Genre
Popularity
Vote Count
Vote Average
Release Date

<h2>Overview</h2>
These fields help in understanding what type of content performs well and how tastes have changed over time.

<h2>Tools & Libraries Used</h2>
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
Analysis Performed

1. Data Cleaning
Removed unnecessary columns
Checked data types
Handled missing values
Converted release date to proper format
Extracted release year for trend analysis

2. Exploratory Visualizations
3. Visual 1: Genre Distribution
Shows which genres dominate the dataset.

Visual 2: Release Year Distribution
Reveals trends in movie production over time.

Visual 3: Vote Count Distribution
Shows how voting is spread among movies.

<h3>Additional Visuals Added-</h3>

Year-wise trend analysis
Vote Count vs Popularity
Popularity by Genre

<h2>Conclusion</h2>

This EDA provides detailed insights into:
Audience preferences
Trending genres
Movie popularity drivers
Industry production timelines

The dataset reveals that:
Drama remains the most dominant and popular genre
2020 saw a spike in movie production
Popularity is strongly tied to vote count
Big-budget action franchises gain top popularity.

These findings can help content creators, marketers, and streaming platforms make informed decisions.
